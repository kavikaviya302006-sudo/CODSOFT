# To-Do List
## Description
This is a simple command-line To-Do List application built using Python.

It allows users to manage daily tasks by adding, viewing, updating, deleting, and clearing tasks through a menu-driven interface.
## Features
* Add new tasks
* View all tasks
* Update existing tasks
* Delete a specific task
* Clear all tasks at once
* Exit the application safely
## Technologies Used
Python
