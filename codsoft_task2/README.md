# Task 2: Calculator 
## Description
This project is a menu-driven Advanced Calculator application developed using Python.

It performs basic and advanced arithmetic operations such as addition, subtraction, multiplication, division, modulus, and power.
## Features
* Addition
* Subtraction
* Multiplication
* Division (with zero-division handling)
* Modulus operation
* Power calculation
* User-friendly menu
* Continuous execution until exit
## Technologies Used
Python 
## How to Run 
1. Ensure Python is installed
2. Clone the repository or download the file.
3. Run the program
