# Task4: Rock-Paper-Scissors
## Description
This project is a command-line Rock-Paper-Scissors game developed using Python.

The user plays against the computer, and the program keeps track of scores until the user decides to exit.
## Features
* User vs Computer gameplay
* Random computer choice
* Score tracking for both player and computer
* User-friendly messages with emojis
## Technologies Used
* Python 
* Built-in module:
random
## How to Run 
1. Check Python installation:
2. Clone the repository or download the file.
3. Run the program:
python rps_game.py
