# Task5: Contact Book
## Description
This project is a menu-driven Contact Book application developed using Python.

It allows users to store, manage, search, update, and delete contact details such as phone number, email, and address.
## Features
* Add new contacts
* View all saved contacts
* Search contacts by name
* Update existing contact details
* Delete contacts
* User-friendly menu system
## Technologies Used
Python
## How to Run 
1. Check Python installation
2. Clone the repository or download the file.
3. Run the program
